Урок 21: Введение в HTML

1. Доктайп и корневой элемент

<!DOCTYPE html> указывает браузеру, что документ использует HTML5. <html lang="ru"> — корневой элемент страницы.

<!DOCTYPE html>
<html lang="ru">
  ...
</html>

2. Секция <head> и метаданные

Содержит метатеги, <title>, подключение стилей/скриптов. Важна для SEO и работы браузера.

<head>
  <meta charset="UTF-8">
  <meta name="description" content="Урок HTML">
  <meta name="keywords" content="HTML, урок">
  <meta name="author" content="Преподаватель">
  <title>Урок 21</title>
</head>

3. Тело документа <body>

Содержит весь видимый контент: текст, изображения, ссылки, формы.

<body>
  <h1>Привет, мир!</h1>
  <p>Текст абзаца.</p>
</body>

4. Заголовки <h1>–<h6>

Уровни заголовков от h1 (самый важный) до h6 (наименее важный).

Заголовок H1

Заголовок H2

Заголовок H3

Заголовок H4

Заголовок H5
Заголовок H6
<h1>Заголовок H1</h1>
<h2>Заголовок H2</h2>
... до H6

5. Параграфы <p>

Отображает абзац текста с отступами.

Первый абзац.

Второй абзац.

<p>Первый абзац.</p>
<p>Второй абзац.</p>

6. Ссылки <a>

Гиперссылки: href — адрес, target="_blank" — открыть в новой вкладке.

Пример ссылки 
<a href="https://example.com" target="_blank">Пример ссылки</a>

7. Изображения <img>

Вставка изображения: src — путь, alt — описание.

<img src="https://via.placeholder.com/150" alt="Пример">

8. Списки: <ul>, <ol>, <dl>

Неупорядоченные (ul), упорядоченные (ol) и описательные списки (dl, dt, dd).

  • Пункт списка 1
  • Пункт списка 2
  1. Первый элемент
  2. Второй элемент
Термин
Описание термина
<ul>
  <li>Пункт списка 1</li>
  <li>Пункт списка 2</li>
</ul>

<ol>
  <li>Первый элемент</li>
  <li>Второй элемент</li>
</ol>

<dl>
  <dt>Термин</dt>
  <dd>Описание термина</dd>
</dl>

9. Формы: <form>, <input>, <textarea>, <select>

Формы для ввода данных. Пример веб-отображения формы:




<form action="#" method="post">
  <input type="text" name="username" placeholder="Имя пользователя">
  <input type="email" name="email" placeholder="Email">
  <textarea name="message">Введите сообщение...</textarea>
  <select name="options">
    <option value="1">Опция 1</option>
    <option value="2">Опция 2</option>
  </select>
</form>

10. Семантические теги: <main>, <aside>, <figure>, <figcaption>

Эти теги улучшают структуру и доступность контента.

Основной контент
Боковая панель
<main>Основной контент</main>
<aside>Боковая панель</aside>
<figure>
  <img src="https://via.placeholder.com/100" alt="Изображение">
  <figcaption>Подпись к изображению</figcaption>
</figure>

11. Блоки и строчные элементы: <div> и <span>

div — блочный контейнер, span — строчный, используются для группировки элементов.

Это блочный div
Это строчный span 
<div style="background:#eee;padding:5px;">Это блочный div</div>
<span style="color:red;">Это строчный span</span>

12. Код: <code> и <pre>

code — для коротких фрагментов кода, pre — для блоков с сохранением форматирования.

Пример inline: console.log()

function test() {
  console.log('Hello');
}
<p>Inline

13. Подвал: <footer>

footer содержит контактную или авторскую информацию внизу страницы.

Контакты: info@example.com | © 2025
<footer>Контакты: info@example.com | © 2025</footer>
